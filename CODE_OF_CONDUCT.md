# Code of Conduct

This project follows a simple standard: be respectful, constructive, and professional.

## Expected Behavior

- Be welcoming and inclusive.
- Focus feedback on code and outcomes.
- Assume positive intent and ask clarifying questions.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Doxxing or sharing private information.
- Trolling, intimidation, or sustained disruption.

## Enforcement

Maintainers may remove content or restrict participation for behavior that violates this code.
